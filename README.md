# 🏗️ Projetos em Destaque

## 🚗 RevendaHub — ERP SaaS para Revendas de Veículos

Sistema completo de gestão para revendedoras de veículos com **arquitetura multi-tenant**.

**Stack**

```
Nuxt 3 · Spring Boot 3 · Supabase · PostgreSQL · Vercel · Render
```

### Módulos implementados

🚗 **Estoque de Veículos**  
Cadastro completo com fotos, documentos e histórico.

👥 **Clientes e Contratos**  
Gestão de compradores/vendedores e geração automática de contratos.

💰 **Financeiro**  
Contas a pagar/receber, fluxo de caixa e relatórios.

🧾 **Notas Fiscais**  
Integração com módulo NF-e via **iframe + SSO automático**.

🔐 **Aprovação de Usuários**  
Fluxo de cadastro com aprovação manual pelo admin.

📋 **Auditoria**  
Log completo de ações por empresa (restrito ao Admin).

🔒 **Multi-tenancy**  
Isolamento total de dados por empresa com **TenantContext**.

### Destaques técnicos

- Autenticação **JWT com Supabase Auth + Spring Security**
- **Clean Architecture** no backend
- **Flyway** para migrations versionadas
- **SSO entre sistemas** via `postMessage + setSession`
- **CI/CD com GitHub Actions**
- Deploy automático no **Render** e **Vercel**

## 🧾 Gestor-NF — Gerenciador de Notas Fiscais

Sistema para **controle e gerenciamento de notas fiscais eletrônicas (NF-e)** com foco em **automação de processos fiscais**.

**Stack**

```
Next · Node · Express · Supabase · PostgreSQL · Vercel 
```

### Funcionalidades

📂 **Emissão de notas fiscais modelo 55 (DANFE)**  
Emissão de notas fiscais em PDF e XML.

📊 **Extração de dados fiscais**  
Identificação de emitente, destinatário, valores, impostos e produtos.

📁 **Organização de documentos fiscais**  
Armazenamento estruturado das notas para consulta e auditoria.

📈 **Controle de emissões e histórico**  
Visualização de notas emitidas, recebidas e relatórios básicos.

### Objetivo do sistema

Criar uma ferramenta que permita **centralizar e organizar notas fiscais automaticamente**, reduzindo trabalho manual e facilitando o controle contábil e financeiro.
